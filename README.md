# Market Risk Management

This repository offers a comprehensive toolkit for market risk management, featuring bootstrapping, Monte Carlo simulation, Value at Risk (VaR) calculation, VaR mapping, and other advanced risk modeling techniques.

## Code Modules

- **Code 1**: Economic state analysis, mean reversion analysis, autocorrelation analysis, and distribution analysis.  
- **Code 2**:
  - Calculate the correlation between two assets using returns data.  
  - Analyze the impact of correlation on portfolio VaR.  
  - Compute realized correlation and swap payoff based on a given correlation matrix and fixed correlation.  
- **Code 3**: Includes three examples demonstrating risk modeling concepts.  
- **Code 4**: Calculation of undiversified and diversified VaR.  
- **Code 5**: VaR mapping techniques.  
- **Code 6**: Expected shortfall, parametric normal, and lognormal VaR calculations.  
- **Code 7**: Python implementation showcasing the differences between bootstrapping and Monte Carlo simulation.  
- **Code 8**: Python demonstration on when to use bootstrapping versus Monte Carlo simulation.
